A commandline client for controlling your Klyqa devices.

Using your klyqa account or dev mode for bulbs onboarded in dev mode.

It is written in python.

You can run the python script directly with


\begin{DoxyCode}{0}
\DoxyCodeLine{./klyqa-\/ctl.py}

\end{DoxyCode}


Use pip for installation of required dependency packages.


\begin{DoxyCode}{0}
\DoxyCodeLine{pip install -\/r requirements.txt}

\end{DoxyCode}


Tested and developed with Python v3.\+9.

or build it as docker container.\hypertarget{md_README_autotoc_md1}{}\doxysection{Build docker python script container}\label{md_README_autotoc_md1}
